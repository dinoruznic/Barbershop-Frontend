# Barbershop Frontend

Modern barbershop management application built with React, TypeScript and Vite.

## Features

- Landing Page
- Authentication
- Appointment Booking
- Client Dashboard
- Employee Dashboard
- Owner Dashboard
- Reviews & Ratings
- Inventory Management

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
